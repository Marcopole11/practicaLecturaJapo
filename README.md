# Opensource simple app to practice basic JP reading

Simple app to practice reading Hiragana and Katakana.

The app allows you to change the format of the text, choosing typography and orientation.
If you're stuck with a character, hover the mouse over it.
Characters you missed or haven't apeared yet have higher chances to appear.
I made this app for my own practice but I feel it can be useful for other users.


I want to keep this free & opensource for the sake of education, use it for free here:

[Spanish version](https://www.mabl.icu/jp/practicajapo.html) | [English version](https://www.mabl.icu/jp/practicejp.html)
